You look at a boomerang in your inventory... Made of wood, this throwing
weapon isn't as damaging as some, but it has the added bonus of flying
back to the hand of its user.

Object 'boomerang' type throwing weapon, extra flags none. Weight 2,
value 1296, level 50. Object Quality (40 / 40 hps) It does 14 to 25
points of damage.

This item seems to be throwing weapon. This item appears meant to be
wielded as a weapon. It will return when thrown, if it hits. It will
return when thrown, if it missess. It would make a poor melee weapon. It
is easy to catch.
