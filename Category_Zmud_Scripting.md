The scripts listed in this category are written to work in the
[zMUD](http://www.zuggsoft.com/page.php?file=zmud/zmudinfo.htm) client.

If you would like to request a script, please do so on the
[Requests](/wiki/Requests "wikilink") page.

If you would like to report a bug or request a feature for a specific
script, please do so on that trigger's talk page.

Scripts
-------

**[Run Stats Counter](/wiki/Zmud_Run_Stats_Counter "wikilink"):** *Counts the
number of kills and amount of experience you have gained on a run and
shows it to the group.*  
**[Bash Counter](/wiki/Zmud_Bash_Counter "wikilink"):** *Counts the number of
times you bash and shows your percentage of successful bashes.*  
**[WikiSearch](/wiki/Zmud_WikiSearch "wikilink"):** *Searches for the selected
text on AvatarWiki when you select and right click it.*  
**Before using triggers, make sure you are familiar with Avatar's
[Trigger Policy](/wiki/Trigger-Using_Policy "wikilink").**
