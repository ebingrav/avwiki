Syntax: cast 'requiem' <target>

Prerequisites: None

You can attempt a prayer for a specific player's corpse with this spell.
By doing so, you can strengthen that player's corpse much like a
physical embalm will do. It can be done anywhere, however.

(You will have poor luck unless you cast it using a player's name. It
will reinforce the timer of the player's heaviest corpse if there is
more than one.)
