[Throwing-weapon](/wiki/Category%3A_Throwing_Weapons "wikilink")-[type](Category%3A_Object_Types "wikilink")
[objects](/wiki/Category%3A_Objects "wikilink") have four
[values](/wiki/Category%3A_Object_Values "wikilink").

-   **Behaviors:** This value determines which set of behavior(s) this
    [throwing weapon](/wiki/Category%3A_Throwing_Weapons "wikilink") will
    exhibit when it is used, which behaviors are (1) it drops to the
    ground when it hits, (2) it disappears when it hits, (3) it
    disappears when it misses, (4) it returns when it hits, (5) it
    returns when it misses, (6) it is easy to catch, (7) it is
    impossible to catch, (8) it "stings" those it hits (increasing its
    chances of conveying any [poison](/wiki/Category%3A_Poisons "wikilink")
    that it may carry), (9) it functions well as a melee weapon, (10) it
    functions poorly as a melee weapon, (11) it does no melee damage,
    and (12) it does area damage.

<!-- -->

-   **Damage, Minimum:** This value determines this [throwing
    weapon](/wiki/Category%3A_Throwing_Weapons "wikilink")'s minimum possible
    damage to those it hits.

<!-- -->

-   **Damage, Maximum:** This value determines this [throwing
    weapon](/wiki/Category%3A_Throwing_Weapons "wikilink")'s maximum possible
    damage to those it hits.

<!-- -->

-   **Poison Status:** This value determines whether or not this
    [throwing weapon](/wiki/Category%3A_Throwing_Weapons "wikilink") may be
    [poisoned](/wiki/Poison_(command) "wikilink") and, if it may, then which
    [type of poison](/wiki/Category%3A_Poisons "wikilink"), if any,
    it carries.

See also [Throwing Weapons](/wiki/Category%3A_Throwing_Weapons "wikilink").
